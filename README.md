Side Scroller Peli (Jump Man)

Pelissä hypitään esteiden yli. Mitä pidemmälle pääset, niin sitä isommat pisteet saat. 

Ohjeet
Hyppy nappi on Spacebar
Mitään muuta ei tarvita paitsi nopeutta ja hyvät refleksit.

Pelin saa päälle painamalla Play painiketta. Jos osut esteeseen, niin voit aloittaa alusta painamalla Restart nappia.
Pelissä ei ole loppua, joten yritä saada hyvät pisteet osumatta esteisiin.
